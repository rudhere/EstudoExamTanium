1.1 Linear Chain Architecture

Arquiteturas Hub-and-Spoke falham em escala massiva (100.000+ endpoints). O Tanium resolve com auto-organização em sub-redes locais: agentes fazem Neighbor Discovery (Camada 2), elegem um Subnet Leader que atua como ponte WAN.

Fórmula de Tráfego

Tráfego WAN Tradicional = N × S (Linear)
Tráfego WAN Tanium = C × S (Constante, C = cadeias lógicas)
1.2 Segurança Criptográfica

Privilégios máximos (SYSTEM/root) protegidos por criptografia de curva elíptica.

tanium-init.dat

Chave pública do Server. Mensagens sem assinatura válida são descartadas na porta 17472.

client.cfg

IPs dos servidores, verbosidade de logs, regras de pareamento local.

Zone Server (DMZ)

Proxy reverso TLS para comunicações externas.

1.3 Conceitos Fundamentais
QuestionsInstruções declarativas que pedem aos endpoints para calcular scripts localmente.
Sensors (Read-only)Scripts de leitura, timeout 60s, sandbox restrito temporário.
Actions & PackagesOrdens de modificação (escrita). Baixa fatias do pacote via cadeia local.
Groups & TagsGrupos Dinâmicos atualizam filiações por regras. Tags são persistentes em arquivos locais.
